# Support

## Getting help

- Usage questions: open a GitHub issue with your command, config shape, and expected behavior.
- Bugs: use the bug report template.
- Security issues: use the private path in [SECURITY.md](./SECURITY.md).

## What to include

- LazyCompass version
- OS and architecture
- MongoDB version if relevant
- Reproduction steps
- Relevant logs or error output with secrets removed

## Scope

- Best-effort support for the latest tagged release
- No guaranteed response time
