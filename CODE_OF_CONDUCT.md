# Code of Conduct

## Our standard

Be respectful, direct, and constructive. Harassment, discrimination, intimidation, or bad-faith disruption are not acceptable.

## Scope

This applies to issues, pull requests, release discussions, and other project spaces.

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, commits, code, or contributions that violate this code of conduct.

## Reporting

Open a minimal GitHub issue requesting maintainer follow-up and avoid posting unnecessary personal details.
